Banco de Dados: MySQL
1. O arquivo MYSQL.txt contêm os dados da criação do schema, tabela e
da stored procedure sp_ins;

Tecnologia de Desenvolvimento:

1. Este aplicativo foi desenvolvido com o Visual Studio 2015 na 
linguagem C#.NET

2. Este aplicativo pode ser executado diretamente no Visual Studio 2015
após a criação das tabelas no MYSQL;

3. O Nugat foi utilizado para adicionar os packages do MYSQL e do
JQuery UI na aplicação;

4. Por se tratar de uma aplicação WEB tenha a certeza de ter instalado
o IIS no Windows da máquina onde irá executar e abrir os fontes desta
aplicação. Conforme dito anteriormente este fontes devem ser abertos
com o Visual Studio 2015;

5. Fui utilizado um servidor de rede com o MYSQL instalado; A string
de conexão existente na abertura do MYSQL devem ser modificadas para
que a aplicação funcione apropriadamente caso deseje utilizar um servidor
MYSQL como localhost. Por favor, lembre-se de criar as tabelas conforme
se encontram no arquivo MYSQL.txt e realizar as mudanças necessárias na
string de conexão para acessar as tabelas.

6. O arquivo style.css foi preparado para esta aplicação para as cores
definidas. Realizei muitos teste de debug para chegar ao resultado esperado.
Os passos seguidos no desenvolvimento foram:

1. Criação de um modelo master page para todas as telas;
2. Criação de um style.css adequado;
3. Criação de um menu para a aplicação;
4. Criação do formulário de cadastro de produtos;
5. Criação do formulário de consulta de produtos;
6. Teste de inserção de dados e consulta de dados.
